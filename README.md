# task-management
## 【作成中】タスク自動化システム
- Visual C++
- Visual Studio 2019 (v142)

## About
- コピー元のファイルの場所をINIファイルから読み込み、コピー先のディレクトリにコピーする。
- （予定）初期設定は、初回起動時に設定可能。設定を後から変更することも可能。その設定は、INIファイルに保存する。
